Project Overview
This project analyzes the Maven Roasters NYC Coffee Shop dataset — 149,000 transactions across three café locations from Jan–Jun 2023.
The goal was to perform end‑to‑end EDA, answer multiple business questions, and uncover store‑level behavioural patterns that aggregated numbers usually hide.

The analysis includes:
Exploratory Data Analysis (EDA)
Statistical testing
Customer behaviour insights
Store‑level performance breakdown
Product‑level performance evaluation
Visual storytelling

Business Questions
Time of day vs transaction volume
Coffee + bakery association
Weekend vs weekday spend
Store location impact on spend
Underperforming products across stores

Key Insights
Hot drinks dominate: 69% of all transactions
Sales doubled from Jan → Jun
Store behaviour differs significantly (p = 0.0000)
Customers pair drinks with drinks, not bakery
Low‑volume items (Coffee Beans, Branded) still drive value
Hell’s Kitchen = consistency, Lower Manhattan = high spend, Astoria = evening strength
